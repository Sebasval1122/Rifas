Interfaz mínima Next.js + TypeScript

Comandos:

```bash
npm install
npm run dev
```

Visitar: http://localhost:3000

Esta implementación es un esqueleto con páginas de ejemplo:
- `/` Dashboard
- `/tickets` Listado de tickets (placeholder)
- `/auth/login` Formulario de login (placeholder)

Siguientes pasos sugeridos:
- Conectar con la API en `README.md` usando `fetch` o `axios`.
- Añadir autenticación (guardar JWT en localStorage).
- Crear componentes y formularios reales.
